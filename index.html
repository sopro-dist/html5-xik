This is the index file for the repository https://github.com/sopro-dist/html5-xik

The app itself is not bundled with SoPro due to size constraints. If you want to use this app, follow these steps:

  1. Clone the repo: 
    git clone https://github.com/SocietyPro/html5-xik.git
  2. Put the new files in your Cambrian Apps directory:
    cp -r html5-xik ~/.Cambrian/Apps
  3. Open SoPro and launch the app through the hamburger menu.
